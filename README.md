# Fomotoken-BSC-Scrapper
A tool can scrape Fomotoken BSC New Token then send to telegram

# Usage:

    1. Install Python 3.9
    
    2. Open CMD on Bot Folder: pip install -r requirements.txt
    
    3. Fill the config.py
    
    4. Run the script
